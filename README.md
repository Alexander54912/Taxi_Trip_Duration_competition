# NYC Taxi Trip Duration — Предсказание длительности поездки такси

Проект по предсказанию длительности поездки такси в Нью-Йорке на основе данных соревнования [Kaggle: New York City Taxi Trip Duration](https://www.kaggle.com/c/nyc-taxi-trip-duration).

## Описание задачи

По данным о поездках такси в Нью-Йорке (координаты посадки/высадки, время, число пассажиров и другие параметры) необходимо предсказать длительность поездки. Метрика качества — **RMSLE** (Root Mean Squared Logarithmic Error).

## Результаты

| Этап | RMSLE (test) |
|------|-------------|
| Baseline (константа) | ~0.79 |
| Временные признаки | 0.5887 |
| + Координаты и расстояние | 0.4855 |
| + Очистка выбросов | 0.4189 |
| Ridge (best alpha, CV) | **0.4185** |

## Подход

### 1. Разведочный анализ данных (EDA)
- Анализ распределения целевой переменной и обоснование логарифмирования
- Обнаружение аномальных периодов (шторм Jonas, метель февраля 2016)
- Визуализация географического распределения поездок

### 2. Feature Engineering
- **Временные признаки:** день недели, час, месяц, бинарные флаги аномалий
- **Координатные признаки:** расстояние гаверсинуса, логарифм расстояния
- **Аэропорты:** бинарные признаки для JFK, LaGuardia, Newark
- **Дорожная ситуация:** признаки пробок (будни 10–18) и свободных дорог (ночь)

### 3. Моделирование
- Ridge-регрессия с подбором alpha через 5-fold кросс-валидацию
- Сравнение Ridge и Lasso: анализ разреженности и trade-off качество/сложность
- Очистка выбросов по остаткам регрессии

## Запуск

### Требования
```bash
pip install -r requirements.txt
```

### Данные
Данные загружаются из Kaggle. Для этого нужен Kaggle API token (`~/.kaggle/kaggle.json`):
```bash
kaggle competitions download -c nyc-taxi-trip-duration
```

### Запуск ноутбука
```bash
jupyter notebook nyc-taxi-trip-duration.ipynb
```
